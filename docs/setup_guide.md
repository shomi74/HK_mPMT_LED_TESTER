# Setup Guide

## Hardware Needed
- Raspberry Pi Pico
- Micro-USB data cable
- Carrier board or breadboard setup
- LED cap or test LEDs

## Serial Monitor
Recommended beginner option: Arduino IDE Serial Monitor.

Settings:
- Baud: 115200
- Line ending: LF

## First Test
Send:
`TEST`

Expected response:
`OK`

## LED Test
Send:
`ON 0 1000`

Then:
`OFF`
